# Samsung S23+ 安裝與測試

## 第一次安裝

1. 以 Samsung S23+ 的 **Chrome** 開啟 GitHub Pages 網址。
2. 點 Chrome 右上角的 `⋮`。
3. 選擇 **安裝應用程式**；若沒有該選項，選 **加到主畫面**。
4. 名稱會是「橙攝器材查價」，確認後桌面會出現圖示。
5. 後續一律點此圖示開啟，不需要重新下載或安裝。

## 建議驗收項目

1. 搜尋 `RF 24-70`，應只看到 Canon RF 24-70mm F2.8L IS USM。
2. 點進品項，確認每日租金、押金、本票金額、公開說明和內附配件皆可閱讀。
3. 點 **在橙攝官方頁面開啟**，應在原始橙攝頁面顯示同一品項。
4. 收藏一項器材，再按首頁右上角 **收藏**，應只顯示已收藏品項。
5. 關閉網路後重開 App，確認最近查看過的資料仍可讀取。
6. 恢復網路後按 **檢查更新**，確認會顯示已發布資料的最後更新時間。

## 每日更新會怎麼出現在手機上

GitHub Actions 在雲端完成更新後，GitHub Pages 會發布新的資料檔。你下次開啟 App 時會先顯示已快取的內容，再檢查是否有新版；發現新版便自動使用。你不需要重新安裝。

若 GitHub 排程遇到繁忙而延後，App 會繼續顯示最後一次成功資料的時間；不會清空資料。

## 租借清單測試

1. 開啟任一器材，在卡片右下角按「＋」，或在詳情頁按「加入清單」。
2. 點右上角或畫面底部固定的「查看租借清單」，用「全體預設租期」的加減按鈕調整天數；畫面會立即更新「基本日租預估」。
3. 需要留存時可直接截圖，或按「列印／另存 PDF」後選擇 Android 的「另存為 PDF」；「下載文字清單」適合傳給同事。
4. 按器材的「官方頁面 ↗」回到橙攝網站登入、確認檔期並完成實際預約。
