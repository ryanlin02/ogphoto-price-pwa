# 租借清單與預估金額功能評估

## 結論

可行，且已完成第一版。功能定位為「租借準備清單與預估金額」，不做官方訂單、不宣稱有庫存，也不將計算結果視為正式報價。

## 已實作的第一版功能

1. 每個品項新增「加入清單」按鈕；清單數量顯示在首頁。
2. 清單頁以緊湊單頁顯示品名、每日租金、押金、本票金額、租借天數、數量、單項小計與官方連結。
3. 可設定全體租借天數，必要時再覆寫個別品項的天數；可移除品項和加入自己的備註。
4. 顯示「基本日租預估」：每日租金 × 天數 × 數量；押金與本票金額另外合計，不和租金混算。
5. 提供「列印／另存 PDF」的單頁版面，適合 Android 的列印功能另存 PDF；同時提供純文字下載。截圖則可直接使用手機原生截圖。
6. 清單保存在此手機瀏覽器，直到使用者手動清除；不傳送任何資料到官方網站。

## 為何不能直接當成官方總價

官方詳情中的長租優惠與個案條件是自由文字，且並非每個品項都有相同規則，例如 3 至 5 天折扣、長租個案報價、不可單租或搭配限制。只用表格的每日租金相乘最透明、也最不會錯誤承諾。

因此清單上必須清楚標示：

> 預估金額依公開「每日租金」計算，未自動套用長租優惠、個案報價、運費、證件／押金規則或庫存狀態；最終租金以橙攝官方確認為準。

## 不建議納入的項目

- 自動替使用者登入或勾選官方預約單。
- 推測庫存、檔期與可租狀態。
- 自動解析所有詳情文字中的優惠並產生正式報價。

## 後續可選擇的第二版

- 輸入租借日期，讓清單自動顯示天數。
- 加入活動名稱、聯絡人、取件提醒與清單備註。
- 匯出更適合傳給同事的 PDF／CSV；仍保留不含個資的預設。
